The **Styles** section contains a set of options that allow you to choose between various styles to customize the appearance of the related posts list. These are global settings, and many of these can be overridden in the widget or the shortcode.

## Related posts style

This allows you to select one of the built-in styles. Besides the first two, i.e. No styles and Text only, the plugin will enqueue the corresponding stylesheet when the posts are displayed.

- No styles: Select this option if you don’t want the plugin to add any styles, but respect the display of the thumbnails. You will need to add your own styles in this case
- Text only: This forces the display of the posts to be text only. i.e. no thumbnails

You can check out the latest version of the [stylesheets on GitHub](https://github.com/WebberZone/contextual-related-posts/tree/master/css).

## Custom CSS

You can add your own CSS here to style the output. Do not include `style` tags. The plugin will not fix your CSS code, so please ensure you included accurate code only.
